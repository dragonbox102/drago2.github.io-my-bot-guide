Grow a Garden Discord Bot Guide

This guide will walk you through the steps to set up and use the Grow a Garden Discord bot on your server.

Step 1: Invite the Bot to Your Discord Server

The first thing you need to do is invite the "Grow a Garden stocks" bot to your Discord server. When you invite the bot, you'll see a screen similar to this, asking for permissions:

Make sure you select the server you want to add it to and confirm the necessary permissions (Add a bot to a server, Create commands).

Step 2: Check Bot Permissions and Setup

Once the bot is in your server, you need to ensure it has all the necessary permissions. Use the /status command to check:

As you can see, the permissions (Manage Roles, Send Messages, Use External Emojis, Embed Links) should all have green checkmarks. If any are missing, you may need to adjust the bot's role permissions in your server settings.

After confirming the permissions, you'll use the /setupbot command. This command will automatically create the necessary roles for seeds, gears, and eggs, which the bot uses for stock updates.

The bot will confirm that the setup is complete and list the existing roles it has processed.

Step 3: Set Up Stock Update Channels

Now, you need to configure the specific channels where the bot will post stock updates. You will use three separate commands for this:

  • Use /setchannel for general seed stock updates.
  • Use /seteggshop for egg shop updates.
  • Use /setgearshop for gear shop updates.

Here's how it looks when you set these channels:

Once you've successfully used these commands, the bot will confirm that each channel has been set.

Step 4: Verify Your Configuration

To ensure you have correctly configured all the channels, you can use the /status command again. This will display a comprehensive overview of the bot's current configuration, including the channels you just set:

Look under "Configured Channels" to confirm that your chosen channels (e.g., #seed-stocks, #gear-stocks, #egg-stocks) are correctly listed. If everything looks good, your Grow a Garden bot is now fully set up and ready to provide stock updates!

Step 5: Get Pinged for Specific Stock Updates!

To receive notifications (pings) for specific item stock updates (like a particular seed, egg, or gear), you'll need to assign yourself the corresponding item's role that the bot created during setup.

For example, if you want to be pinged when "Grape" seeds are in stock, you need to get the "Grape" role.

How to Get Specific Item Roles:

Your Discord server might have "Customisation Questions" or a similar onboarding process to help you select roles:

Or you might find a dedicated "roles" channel where you can click reactions to gain roles for various items:

Look for roles named after specific seeds (e.g., "Pepper Seed", "Cacao"), eggs, or gears. Assign yourself the roles for the items you want to be notified about! If your server has Discord's built-in **Onboarding**, it's usually the easiest way to pick your preferred roles.

Super Simple Grow a Garden Bot Guide for Everyone!

This is the easiest guide to get your Grow a Garden bot working. Just follow these steps!

1. Invite the Bot

Get the bot into your Discord server. An admin needs to click "Invite" and pick the server.

2. Set Up the Bot

In Discord, type /setupbot and hit enter. This makes important roles.

Then, type /status to check if everything looks good (permissions should be green checks).

3. Tell Bot Where to Post

Tell the bot which channels to use for updates:

  • For general seed updates: Type /setchannel (and pick a channel).
  • For egg shop updates: Type /seteggshop (and pick a channel).
  • For gear shop updates: Type /setgearshop (and pick a channel).

4. All Done!

Type /status one more time to see your setup. Your channels should be listed!

5. Get Pings!

If you want notifications for specific seeds, eggs, or gears, you need to get their special roles.
